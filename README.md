## What's Inside

Every breakdown includes:

- ✅ **Hook Analysis** — why the first 3 seconds worked
- ✅ **Retention Mechanics** — what kept viewers watching
- ✅ **Mind Map** — visual structure of the script (Mermaid, renders on GitHub)
- ✅ **Replicable Formula** — a 2–3 sentence template you can adapt
- ✅ **Partial Transcript** (first 60%, auto-generated)
- ✅ **Original Video Link** with full creator attribution

## How These Breakdowns Are Made

We use [TokTranscript](https://toktranscript.com/?utm_source=github&utm_medium=readme&utm_campaign=hero) to extract transcripts and viral signals from public TikTok videos. Each breakdown is then enriched with hook analysis and Mermaid mind maps using LLMs.
