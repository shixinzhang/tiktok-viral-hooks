## 拆解流程

我们从 [TokTranscript](https://toktranscript.com/?utm_source=github&utm_medium=readme&utm_campaign=method) 拉取公开的文稿和互动数据,然后用 4 个固定问题审视每个视频:

1. **前 3 秒承诺了什么?**
2. **大约第 7 秒,什么模式打断让观众继续看?**
3. **为什么这条片被算法从当天数百万视频里挑出来推?**
4. **新手能直接抄的一句话公式是什么?**

答案被渲染成你正在看的 markdown 页面。每天 [GitHub Action](.github/workflows/daily-publish.yml) 自动更新。
