# All of Algebra in One Minute

## Exponent Multiplication
### Same base multiplication
- Add the exponents
- Example: A² × A³ = A⁵

### Coefficient and base multiplication
- Multiply coefficients first
- Add exponents of same base
- Example: 2A⁵ × 3A = 6A⁶

### Power of a power
- Multiply the exponents
- Example: (A³)² = A⁶

### Power of a product
- Distribute the exponent to each factor
- Example: (2A)² = 4A²

## Exponent Addition
### Same base addition
- Combine coefficients only
- Keep the exponent unchanged
- Example: 3A² + 2A² = 5A²

## Exponent Division
### Same base division
- Divide coefficients
- Subtract exponents
- Example: 4A² ÷ 2A = 2A¹

## Zero Exponent Rule
### Any base to the zero power
- Equals 1
- Example: 3 × A⁰ = 3 × 1 = 3
